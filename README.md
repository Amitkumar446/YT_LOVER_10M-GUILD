## 🚀 Setup Guide

### 1. Clone Project

```bash
git clone https://github.com/your-username/yt-lover-esports.git
cd yt-lover-esports
```

---

### 2. Setup Firebase

1. Go to Firebase Console
2. Create new project
3. Enable:

   * Realtime Database
   * Storage
4. Copy config

Replace in code:

```javascript
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  databaseURL: "YOUR_DB_URL",
  projectId: "YOUR_PROJECT",
  storageBucket: "YOUR_PROJECT.appspot.com",
};
```

---

### 3. Run Project

Simply open:

```
index.html
```

OR use Live Server for best experience
